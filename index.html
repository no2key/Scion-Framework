Scion Framework

Scion Framwork is a powerful PHP5.5+ framework that helps you quickly write simple yet powerful web applications and APIs. Scion is easy to use for both beginners and professionals. This framework use the latest features of PHP.

Features

System Requirements

You need PHP >= 5.5.0 because this framework use:

  • Namespaces
  • Traits
  • Short array syntax
  • Hash function hash_pbkdf2
  • Password hashing function password_hash
